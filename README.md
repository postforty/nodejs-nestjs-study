# "Node.js 백엔드 개발자 되기" 스터디
## http 서버
+ curl
  + 다양한 네트워크 프로토콜로 데이터를 주고받는 프로그램
  + GET, POST, DELETE, PUT 요청
  + 설치: https://curl.se/windows/
+ K6
  + 성능 테스트 도구
  + 자바스크립트로 부하 테스트 시나리오를 만들 수 있음
  + 설치: https://dl.k6.io/msi/
  + 실행: k6 run test_hello.js
